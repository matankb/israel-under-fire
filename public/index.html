Each time a rocket is fired into Israel, sirens go off throughout the country, and thousand of civilians - men, women, and children - are forced to run for their lives to safety.
How is this data collected?
This website uses official data collected from Pikud HaOref (the Israeli Home Front Command), updated every ten minutes. Since Pikud HaOref only supplies data about the previous four hundred sirens, this website stores a list of every past siren since {{ COLLECTION_BEGIN_DATE }} as well, and combines that data with the new data (removing duplicates) to create the final count.
